# HJ Resource Management

电机资源管理完整 demo。当前版本聚焦电机建档、自动编码、标签展示、电脑端管理、手机端扫码查询、手机端入库/出库和操作留痕。

## 运行

```powershell
npm install
npm run prisma:generate
npm run db:init
npm run seed
npm run dev
```

默认开发服务端口为 `4011`，并监听局域网地址。电脑访问：

`http://localhost:4011/login`

手机访问时不能使用 `localhost`，需要使用电脑当前的局域网 IP。IP 可能会在重新联网、切换 Wi-Fi 或路由器重新分配后变化，因此不要长期写死。双击 `tools/启动资源管理系统/启动资源管理系统.exe` 后，启动器会显示当前可用的手机访问地址。

格式如下：

- 登录页：`http://电脑当前局域网IP:4011/login`
- 手机现场入口：`http://电脑当前局域网IP:4011/mobile`

## 默认账号

- 管理员：`admin` / `admin123`
- 查看账号：`viewer` / `viewer123`

## 当前范围

- 登录
- 首页概览
- 电机列表、照片缩略图和搜索
- 新建电机档案，并按型号自动生成唯一编码，例如 `GM6020-0001`
- 第一次建档可上传照片，照片用于列表和详情页辅助识别
- 详情页展示二维码和 Code128 条形码预览
- 详情页可复制编码
- 独立标签打印页
- 电脑端扫码/输入编码入库
- 电脑端扫码/输入编码出库，登记出库人和使用车辆
- 手机端入口
- 手机端扫码/输入编码查询
- 手机端扫码/输入编码入库
- 手机端扫码/输入编码出库
- 操作记录保留建档、入库、出库历史

## 暂不包含

- 微信小程序
- 打标机直连
- 电机编辑、删除、作废
- 数据导出和自动备份
- 完整资源管理平台扩展
