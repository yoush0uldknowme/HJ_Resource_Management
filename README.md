# HJ Resource Management

电机资源管理试点系统。第一版聚焦电机建档、自动生成唯一编码、二维码展示、扫码入库、扫码出库、列表查找、照片展示和操作留痕。

## 运行

```powershell
npm install
npm run prisma:generate
npm run db:init
npm run seed
npm run dev
```

默认开发服务端口固定为 `4011`，并监听局域网地址。电脑访问：

`http://localhost:4011/login`

手机访问时，请把 `localhost` 换成电脑的局域网 IP，例如：

`http://10.151.12.38:4011/login`

## 默认账号

- 管理员：`admin` / `admin123`
- 查看账号：`viewer` / `viewer123`

## 当前范围

- 登录
- 首页概览
- 电机列表、照片缩略图和搜索
- 新建电机档案并自动生成 `型号-递增编号`，例如 `GM6020-0001`
- 第一次建档上传照片，照片用于列表和详情页辨认
- 详情页展示二维码
- 详情页展示 Code128 条形码预览，并可复制编码到打标机本地软件
- 入库：扫码或输入编码后直接入库，不再要求再次拍照
- 出库：扫码或输入编码后登记出库人和使用车辆
- 操作记录保留建档、入库、出库历史
