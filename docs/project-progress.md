# 电机资源管理试点系统进度表

本文档用于记录项目整体计划、当前进度和后续待办。每完成一部分，就在这里更新状态，避免需求和实现过程混乱。

## 当前阶段

当前项目处于：**第一版 MVP 可试跑阶段**。

已经完成主流程：

`首次建档 -> 自动生成唯一编码/二维码 -> 照片辅助识别 -> 扫码入库 -> 扫码出库 -> 查询 -> 操作留痕`

当前试用地址：

`http://localhost:4011/login`

默认账号：

- 管理员：`admin / admin123`
- 查看账号：`viewer / viewer123`

## 总体目标

先完成电机管理试点，不扩展完整资源管理平台。第一阶段只验证电机管理流程是否真的能在现场跑通。

核心目标：

- 每台电机首次购入时建档
- 系统自动生成唯一 SN/内部编码
- 基于唯一编码生成二维码或条形码
- 照片仅用于列表和详情页辅助识别
- 入库以扫码为主，扫码后直接入库
- 出库以扫码为主，只登记出库人和使用车辆
- 所有关键操作保留历史记录
